 Certificate
============================================================================= 
Usage: quik cert [options] <FQDN.domain.name>
 This command was designed for your development and staging servers. It\s import to test
 your installation with an SSL certificate in place. You can choose to self sign your cert
 or create a certificate request signature (csr) file to provide to your hosting provider.

Options:
  -h, --help      Show this message
  -q, --quiet     Command enters non-interactive mode and completes on it's own.
  --csr           Do not self sign the certificate, only produce the certificate request file.
